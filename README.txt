# Дока

Дока — это **добрая** энциклопедия для веб-разработчиков.
Вы можете [открыть её](https://doka.guide) в своём браузере.

## Преимущества

- Понятный материал;
- Нескучные объяснения;
- Удобный поиск.

Проверяется работа алгоритмов сортировки и скорость их работы
Реализация с декоратором и без
с декоратором работают быстро все алгоритмы
без декоратора скорсть сильно отличается
